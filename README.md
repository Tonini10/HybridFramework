# HybridFramework 🧪

Framework híbrido de pruebas de software estructurado en 7 secciones complementarias.
Integra técnicas avanzadas de testing —desde pruebas unitarias BDD hasta modelos predictivos
de confiabilidad— implementadas en JavaScript con Jasmine como motor de pruebas.

## 📊 Resultados
- ✅ 59 specs distribuidas en 7 archivos de prueba
- ✅ 0 failures en la ejecución completa
- ✅ 6 módulos fuente | 7 archivos spec | 2 archivos de configuración

## 📁 Estructura
HybridFramework/

├── spec/         # Archivos de prueba Jasmine
